# Pitfall: Searching for "Google AntiGravity"
Searching for "Google AntiGravity" often returns results about the classic "Google Gravity" Easter Egg by mrdoob. 

If the user mentions "Google AntiGravity" as a serious technical project or product, do not assume it is the Easter Egg. 

1. **Ask for clarification/source**: Explicitly ask for the URL, patent number, or documentation link before searching.
2. **Avoid assumption**: Do not rely on search engine auto-correction or "did you mean" results for this specific term.
3. **Verify claims**: If the user insists on a project, prioritize direct source analysis over broad web search to avoid hallucinating based on the Easter Egg association.
